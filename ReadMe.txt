Инструкция по установке СКНО:

Для запуска, откройте терминал  из каталога Apteki_SKNO_install(не имеет значения где находится сам каталог)
Напишите команду python3 Install_skno.py
Далее следуйте инструкциям которые будут появляться на экране.
В процессе установки будет создан бэкап с файлами , он сохраняется в /home/cashier/.wine/drive_c/Auto-BackUP
